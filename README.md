# calculator
A python calculator app

## 1. Clone / create the folder structure
git clone https://github.com/gbowne1/calculator.git
cd calculator

## 2. (Optional) install as a package
pip install .

## 3. Interactive REPL
python run_calculator.py        # or just `calc` if installed

## 4. One-shot calculation
python run_calculator.py + 7 4
